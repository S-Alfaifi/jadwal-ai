## üìå Project Idea

The goal of this project is to make the process of building a university schedule easier and faster.  
Students can:

1. Add their courses  
2. Add sections for each course (lecture/lab)  
3. Enable or disable specific sections  
4. Generate schedules automatically  

The system checks for:

- **Time conflicts** between classes  
- **Final exam conflicts**  
- **Overlapping sections**

If everything is valid, the app displays all possible schedules that the student can select from.
